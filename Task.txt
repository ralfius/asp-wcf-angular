Написать ASP.NET MVC проект по поиску и редактированию пользователей с следующим функционалом:
- Пользователь характеризуется такими атрибутами:
  а) Имя (минимум 2 символа)
  б) Фамилия (минимум 2 символа)
  в) почта (уникальное значение среди пользователей, формат почты должен быть валидный например:"test@gmail.com" valid, "asdasdasd" invalid)
- Поиск по имени (включая частичное совпадение, например при поиске "ма" выводятся все пользователи у которых встречается "ма" в имени Дима, Максим и тд)
- Поиск по фамилии (включая частичное совпадение)
- Поиск по почте (включая частичное совпадение)
- Возможность отображения всех пользователей в системе если фильтр не задан ни по одному из параметров поиска
- Имплементировать пейджинг для отображения результатов поиска (по умолчанию 10 записей)
- Возможность добавления нового пользователя
- Возможность удаления пользователя
- Возможность редактирования пользователя
- Обработка ошибок (ввода, попытка использовать существующую почту для нового пользователя и тд)
- В качестве дата сорса для пользователей self hosted WCF service (в консоле в соседнем проекте в том же солюшене), данные по пользователям можно хранить в памяти
* Все должно запускать по F5 в VisualStudio
